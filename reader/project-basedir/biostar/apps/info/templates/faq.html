Origins

This website's source code (ln-central) is a software fork of Biostar-central. Biostar-central is a forum focusing on bioinformatics, computational genomics and biological data analysis. The fork was done to better serve the the lightning network community. For example, ln-central has been integrated to use lightning network micro-payments instead of traditional single sign-on used by biostar.

Biostar-central has been developed as an open source software and has been released with the MIT license thanks to the support from the following institutions:

  • The Pennsylvania State University
  • National Institutes of Health (NIH), specifically grant NIH 5R25HG006243-02, Analyzing Next Generation Sequencing Data principal investigator: Titus Brown (Michigan State University)

Parnell LD, Lindenbaum P, Shameer K, Dall'Olio GM, Swan DC, et al. 2011 BioStar: An Online Question & Answer Resource for the Bioinformatics Community. PLoS Comput Biol 7(10): e1002216. doi:10.1371/journal.pcbi.1002216

Biostar was developed in 2011. It can be found at biostars.org
