What happens when I publish this post?

The post will be published after a payment is made on the Lightning Network. The following information is going to be associated with the post and made publicly visible.
  • Your wallet's pubkey
  • A "pen name" base on the pubkey
  • An avatar image base on the pubkey
  • Statistics such as number of posts, up-votes, and reputation score

What if I want to save draft and resume later?

To save current work and continue later, then bookmark this page. The URL of this page encodes the work in-progress. The URL encodes only what's visible on this page. Specifically, the content of the post and timestamp. It does not contain your wallet's pubkey, IP address, or any other data about the person or people involved in writing the post.

How can I know what information is actually stored in the URL?

You can decode the URL with python3 like so
echo PUT_URL_HERE | python3 -c 'import zlib, binascii, urllib.parse, sys; urllib; url = sys.stdin.readline(); memo = urllib.parse.unquote(url.split("/", 6)[6].rstrip("/")); decoded = zlib.decompress(binascii.a2b_base64(memo)).decode("utf-8"); print(decoded)' | python3 -m json.tool

How can I know what information is actually stored in the invoice?

You can decode the payment invoice with Lightning CLI like so
lncli decodepayreq PUT_INVOICE_HERE
